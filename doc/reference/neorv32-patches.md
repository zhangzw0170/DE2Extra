# NEORV32 Submodule Local Patches

> NEORV32 submodule pinned at **v1.13.1** (release tag).
> This document tracks all local modifications to the submodule.
> When upgrading NEORV32 to a new release, re-apply each patch listed below.

## Patch List

### 1. Bootloader UART Baud Rate: 19200 → 115200

**File**: `neorv32/sw/bootloader/config.h`

**Location**: line 34

**Change**:
```c
// before
#define UART_BAUD 19200

// after
#define UART_BAUD 115200
```

**Reason**: Default 19200 is unnecessarily slow at 50MHz CPU clock. 115200 reduces firmware upload time from ~26s to ~4s for a 50KB executable.

**How to re-apply after NEORV32 upgrade**:
1. Check if upstream changed `config.h` — if upstream added a different baud or restructured the config, merge carefully.
2. If `UART_BAUD` still defaults to 19200, change to 115200.
3. Rebuild bootloader: `cd neorv32/sw/bootloader && make bootloader NEORV32_HOME=../..`
4. Re-synthesize FPGA (the bootloader image is baked into the boot ROM VHDL).

**Note**: This patch requires rebuilding the bootloader and re-synthesizing the FPGA bitstream. The `neorv32_imem_image.vhd` (application firmware image) is auto-generated and NOT tracked here.

### 2. (Auto-generated) IMEM Image

**File**: `neorv32/rtl/core/neorv32_imem_image.vhd`

**Nature**: Auto-generated by `make image` from application firmware. Always differs from the upstream v1.13.1 tag.

**Action on upgrade**: No action needed — this file is regenerated every time firmware is compiled.

## Upgrade Checklist

When updating NEORV32 to a new release:

1. `cd neorv32 && git fetch --tags && git checkout <new-tag>`
2. Check if upstream `sw/bootloader/config.h` changed — merge if needed.
3. Re-apply patch #1 (UART_BAUD = 115200).
4. Rebuild bootloader: `cd sw/bootloader && make bootloader NEORV32_HOME=../..`
5. Rebuild application firmware: `cd sw/app/de2shell && make image NEORV32_HOME=../../../neorv32`
6. Re-synthesize FPGA in Quartus.
7. Board test: verify bootloader prompt appears at 115200 baud, firmware uploads and runs correctly.
