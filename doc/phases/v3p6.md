## 1. PS/2 虚拟键码 (Virtual Keycodes)

### 背景

PS/2 解码器 (`ps2_decoder.c`) 原来只输出 ASCII 可打印字符。方向键、F1-F12 等扩展键虽然解码了 scancode，但 `ascii=0, has_ascii=0`，被 `main.c` 的门控丢弃，无法到达用户程序。

### 改动

**`ps2_decoder.h`** — 新增 22 个虚拟键码常量 (0x80+):

| 范围 | 键 | 键码 |
|------|-----|------|
| F1-F12 | 功能键 | 0x80-0x8B |
| 方向键 | UP/DOWN/LEFT/RIGHT | 0x90-0x93 |
| 导航键 | HOME/END/PGUP/PGDN/INS/MENU | 0x94-0x99 |

**`ps2_decoder.c`** — 新增 F1-F12 scancode 映射 (Set 2):
- F1(0x05), F2(0x06), F3(0x04), F4(0x0C), F5(0x03), F6(0x0B)
- F7(0x83), F8(0x0A), F9(0x01), F10(0x09), F11(0x78), F12(0x07)
- 方向键/导航键改为输出虚拟键码 (`ascii=VK, has_ascii=0`)

**`main.c`** — 门控改为 `key.has_ascii || key.ascii != 0`，虚拟键码可通过。

### 使用方式

程序 input handler 中用 `(uint8_t)c` 与 `PS2_VK_xxx` 比较：
```c
switch ((uint8_t)c) {
    case 'w': move_up(); break;
    case PS2_VK_UP: move_up(); break;
    case PS2_VK_F10: quit(); break;
}
```

## 2. 全局 Fn 键绑定

在 `t_active_prog()` 中实现两个全局快捷键，所有程序生效：

| 键 | 功能 | 实现位置 |
|----|------|---------|
| **F1** | 显示当前程序帮助 (name + help → UART) | `prog->input()` 之前拦截 |
| **F10** | 退出当前程序 | `prog->input()` 之后检查，允许程序先做清理 |
| **ESC** | 退出当前程序 (UART 兼容) | 与 F10 同级 |

### 设计决策

- F10 在 `prog->input()` 之后检查，这样需要清理的程序（chroma 禁用硬件、synth 静音）可以在自己的 input handler 中处理 F10
- F1 在 `prog->input()` 之前拦截，程序不会收到 F1

### Q 键退出已全面移除

所有程序不再响应 Q 键退出：

| 程序 | Q 退出 | F10 退出方式 |
|------|--------|-------------|
| hello | 已移除 | 全局 t_active_prog |
| info | 已移除 | 全局 |
| life | 已移除 | 全局 |
| conway_hw | 已移除 | 全局 |
| memtest | 已移除 | 全局 |
| monitor | 已移除 | 全局 |
| dashboard | 已移除 | 全局 |
| demo | 已移除 | 全局 |
| snake | 已移除 | 全局 + 程序内 |
| chroma | 已移除 | 全局 + 程序内（HW 清理） |
| ps2 | 已移除 | 程序内（直读 PS/2） |
| pong_hw | 已移除 | 程序内（直读 PS/2） |
| synth | 保留 Q + 加 F10 | 程序内（直读 PS/2，Q 不在 WASD 旁） |
| twm | Q=关闭窗口（非退出） | ESC 退出 |

## 3. Snake 双人模式

完全重写 `sw/app/de2shell/snake.c` (291 行 → 312 行):

| 特性 | 说明 |
|------|------|
| 模式选择 | 先选 1P/2P → 再选难度 (两页选择) |
| P1 控制 | WASD (绿色 `O`=`o` 头+身) |
| P2 控制 | 方向键 (青色 `#`=`=` 头+身) |
| 1P 兼容 | 单人模式 WASD + 方向键都能用 |
| 共享食物 | `@` 红色，谁先吃到谁得分 |
| 碰撞规则 | 自撞/互撞/头碰头 (双死=平局) |
| 退出键 | F10 (替代 Q，避免 WASD 误触) |
| 速度 | 双人模式按总得分加速 |

### 起始位置

- P1: 左侧 (x=19, y=13)，向右
- P2: 右侧 (x=58, y=13)，向左

### 结束画面

- 1P: `GAME OVER`
- 2P: `P1 WINS!` / `P2 WINS!` / `DRAW!`
- 通用: `R/Space=retry  F10=quit`
