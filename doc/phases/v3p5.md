## 1. 概述

ChromaShader 是基于硬件哈希噪声的 VGA 地形沙盒。80x25 地形区域由 RTL 填充 BRAM（~40µs/帧），C 驱动仅做输入/HUD，地形渲染零 CPU 开销。

### 核心架构

- **Hash 函数**: 6 级 XOR-shift 组合逻辑，32-bit 输出
- **7 种地形**: 深水/浅水/沙滩/草地/森林/山岩/雪峰 (hash[7:0] 阈值划分)
- **3 块 BRAM**: terrain_ram (16-bit fg+bg) + meta_ram (8-bit gold+type) + paint_ram (18-bit valid+gold+fg+bg)
- **VGA 区域覆盖**: 仅对 terminal addr 160-2159 (rows 2-26) 生效，rows 0-1/27-29 由 C driver 写 HUD

### 实际资源 (Quartus 待编译)

| 模块 | 估算 LEs | 估算 M9K |
|------|---------|---------|
| Hash + FSM + 寄存器 | ~200 | 0 |
| terrain_ram (2048x16) | 0 | 4 |
| meta_ram (2048x8) | 0 | 2 |
| paint_ram (2048x18) | 0 | 4 |
| VGA 输出 MUX + RGB332→565 | ~60 | 0 |
| **合计** | **~260** | **~10** |

## 2. 已完成

### RTL (`src/rtl/periph/chroma_shader.vhd`, ~450 行)

- Wishbone slave (50MHz): 9 寄存器 (CTRL/SEED/OFF_X/OFF_Y/PLAYER_X/PLAYER_Y/CELL/PAINT/STATUS)
- Fill FSM: S_IDLE→S_HASH→S_WRITE→S_DONE, 2 cycles/cell = 4000 cycles = 80µs
- VGA output (25MHz): region check + BRAM read + paint mux + player/gold overlay + RGB332→565
- 三块 M9K 推断 RAM，`no_rw_check` pragma

### 总线集成

- **地址**: `0xF0014000` (s12 in wb_intercon)
- `de2extra_pkg.vhd`: `ADDR_CHROMA_BASE`
- `wb_intercon.vhd`: s12 slave port + chip select + response mux
- `de2os_top.vhd`: 实例化 + Wishbone + VGA 侧连接

### VGA 终端集成 (`vga_text_terminal.vhd`)

- 新增 6 个输入端口: `chroma_en_i`, `chroma_char_i`, `chroma_fg_i`, `chroma_bg_i`, `clk_25m_o`, `brm_rd_addr_o`
- 渲染 MUX: `chroma_en_i='1'` 时覆盖 char/fg/bg，抑制光标

### C 驱动 (`sw/app/de2shell_rtos/chroma.c`)

- MMIO 寄存器直接操作 (CHROMA_BASE 0xF0014000)
- `chroma_task_init()`: SEED + 偏移 + 玩家位置 + 使能 + HUD 绘制
- `chroma_task_input()`: WASD→刷新_地形, E→CELL 读取+金矿采集, 1/2/3→PAINT, Q→禁用
- HUD: rows 0-1 标题/帮助, row 27 金矿计数/坐标, row 28 边框, row 29 不动 (系统状态栏)
- `LOCAL_BUILD` 路径保留 (纯软件 ANSI + SDL2)

### RTOS 集成

- `makefile`: 移除 chroma.c filter-out
- `main.c`: PROG_CHROMA 枚举 + `prog_chroma` 注册 + CLI `chroma` 命令
- 栈大小 640 字

### 修复的 Bug

| 问题 | 根因 | 修复 |
|------|------|------|
| meta_ram 全为 metavalue | 两个时钟进程驱动同一信号 (多驱动冲突) | 将 paint 进程的 meta_ram 写入移到 WB 主进程 |
| rgb332_to_565 宽度错误 | g6: 3+2=5≠6, b5: 2+1+1=4≠5 | g6=g3&g3 (3+3=6), b5=b2&b2&b2(1) (2+2+1=5) |
| vopt 段错误 | 内联优化大初始化数组崩溃 | vopt +acc 阻止内联 |
